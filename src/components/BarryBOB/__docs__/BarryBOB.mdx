# BarryBOB

Componente que muestra una tarjeta con un avatar, un título y una descripción.

#### Ejemplo

<Canvas of={BarryBOB.Primary} />

## Uso

```tsx
import BarryBOB from 'milibreria';

function Example() {
  return (
    <BarryBOB
      title="Barry Allen"
      description="The Fastest Man Alive"
      avatar="https://example.com/barry-avatar.jpg"
    />
  );
}

export default Example;
