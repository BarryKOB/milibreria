# MyButton

Componente de botón personalizado con varias propiedades para la personalización.

#### Example - Primary

<Canvas of={MyButtonStories.Primary} />

## Usage

```tsx
import { MyButton } from 'milibreria';

function Example() {
  return (
    <MyButton
      text="Button primary"
      txtcolor="black"
      bgcolor="red"
      bordercolor="red"
      bordersize="2px"
      hoverTxtColor="white"
      hoverBgColor="blue"
      disabled={false}
      size="small"
      onClick={() => alert("Button Primary")}
    />
  );
}

export default Example;
